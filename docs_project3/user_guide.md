### User Guide

#### 1. Roles

-   **User:** Can submit tickets, view their own history, and view their personal metrics dashboard.
    
-   **Agent (Staff):** Can be assigned tickets, change ticket status, and view the "My Tickets" dashboard.
    
-   **Sysadmin:** Can manage Users, Teams, and view global System Performance metrics.
    

#### 2. For Regular Users

1.  **Logging In:** Click "Login with Google" on the homepage.
    
2.  **Creating a Ticket:** Navigate to "Tickets" -> "New Ticket".
    
3.  **Tracking Status:**
    
    -   Go to "My Tickets Dashboard" to see Open vs Resolved counts.
        
    -   You will receive email notifications when an Agent updates your ticket.
        

#### 3. For Agents (Staff)

1.  **Finding Work:**
    
    -   Use the **Board View** to see tickets arranged by status (Kanban style).
        
    -   Use the **Search Bar** to find tickets by keyword (Subject or Description).
        
2.  **Managing Tickets:**
    
    -   **Assign:** You can reassign tickets to other Team members via the dropdown on the ticket show page.
        
    -   **Approve/Reject:** For tickets requiring authorization, use the Approval buttons. _Note: Rejection requires a reason_.
        

#### 4. For Sysadmins

1.  **User Management:** Navigate to the "Users" tab to view all registered accounts. You can Delete users or Edit their roles.
    
2.  **System Analytics:** Access the "Admin Dashboard" to view:
    
    -   Total Ticket Volume.
        
    -   Average Resolution Time.
        
    -   Pie Chart of Tickets by Category.
